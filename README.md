# Flask Web开发 

> 代码基于狗书第二版,运行之前添加.env文件
```.env
MAIL_SERVER=xxx
MAIL_PORT=xxx
MAIL_USERNAME=xxx
MAIL_PASSWORD=xxx
SECRET_KEY=xxx
```

## Installation

```
$ git clone https://github.com/prince5460/my_flasky.git

$ cd my_flasky

$ pipenv install --dev

$ pipenv shell

$ flask deploy

$ flask run

* Running on http://127.0.0.1:5000/
```
